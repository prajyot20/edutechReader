Step to use Edutect Reader 

1. Browse the file from your device by clikcing on browse button (light yellow color button on top left of screen).

2. Once you see the selected file in browse option by clicking on open button you can view the file.


3. Recently viewed file will come on main screen.


4. To clear recently viewed file list you can click on setting and then  on clear lsit option.

5. You can also open any supported file  in edutect reader simple by clicking on file from any location and you will find open with edutech reader as a option (if you dont get that option please go to settings and remove default open with option).

 

Some useful features:

  1. Interactive document reader.
  2. Highlight, Underline, Scratch and annotation option in the file. 
  3. Transalte word to different languages.
  4. Find word anywhere in the document.
  5. Dictionary find meaning of sleected word from the document.
  6. Flowref option to read book in easy way.
